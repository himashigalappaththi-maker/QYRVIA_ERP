# Backend API Inventory (Phase 35)

Generated by static audit of `server/src`. Mount prefixes from `server/src/routes/api.js`.
All `/api/*` groups except `/health` and `/auth` run behind the protected chain
(`authentication → identityContext → businessDate`). Permission codes are the
`requirePermission(...)` guard on each route.

Legend: **C** = command (write via bus), **Q** = query (read via bus), raw = inline handler.
